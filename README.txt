Features:
1. Mouse Picking for Cubes -> Implemented as buttons.
2. Aliasing.
3. Backface Culling.
4. Transparent Water.
5. Fog shader on all objects.
6. Scissor Test.
7. Stencil Test.
8. Reset button.
9. Free moving camera.

Controls:
1. Move mouse to select the buttons.
2. Left Mouse Button to click on buttons.
3. Click 'R' once to go into free moving camera mode.
4. Click 'R' again to reset scene and go back to static camera mode.
